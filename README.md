# 🏡 StaySphere - Intelligent Homestay Booking API

A production-ready REST API built with ASP.NET Core

## 🌐 Live API
https://staysphere-api.onrender.com

## 🛠️ Tech Stack
- ASP.NET Core Web API (.NET 10)
- Entity Framework Core
- PostgreSQL (Supabase)
- JWT Authentication
- BCrypt Password Hashing
- Docker
- Deployed on Render

## 📌 API Endpoints
| Method | Endpoint | Description | Auth |
|--------|----------|-------------|------|
| GET | /api/homestays | Get all homestays | Public |
| GET | /api/homestays/{id} | Get by ID | Public |
| POST | /api/homestays | Create homestay | JWT Required |
| DELETE | /api/homestays/{id} | Delete homestay | Admin Only |
| POST | /api/auth/register | Register user | Public |
| POST | /api/auth/login | Login + get token | Public |

## 🔐 Features
- JWT Authentication
- Role-Based Authorization (Admin/User)
- Input Validation
- Global Exception Handling
- Filtering & Searching
- Docker containerized
- Cloud deployed
